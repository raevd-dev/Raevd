# Raevd

This project uses Vite + React + TypeScript and should be managed with `pnpm`.

## Prerequisites

- Node.js 20+ (recommended)
- `pnpm` installed globally

If you do not have pnpm yet:

```bash
npm install -g pnpm
```

## Install Dependencies

```bash
pnpm install
```

## Run Locally

```bash
pnpm dev
```

Then open the local URL shown in the terminal (usually `http://localhost:5173`).

## Build for Production

```bash
pnpm build
```

## Preview Production Build

```bash
pnpm preview
```

## Lint

```bash
pnpm lint
```

## Format

```bash
pnpm format
```
